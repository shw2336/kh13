# Markdown 사용 설명서

**markdown**이란 **markup**에 반대 개념으로 최소한의 기호로 문서를 꾸밀 수 있도록 고안된 문법 체계이다.

# 제목 작성법
제목은 #이라는 기호를 이용하여 작성할 수 있다.  
개수는 1개부터 6개까지 사용할 수 있다.
일반적으로 1개부터 3개까지만 제목으로 쓰고 4개부터는 일반 글자보다 작아서 잘 사용하지 않는다.


# 대제목( #1개)
## 중제목(# 2개)
### 소제목(# 3개)

#코드 블록

```
일반적인 블록
```
